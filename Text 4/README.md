# Student Performance Prediction using Machine Learning
## Project Overview
This project predicts student final grade using machine learning algorithms.
## Dataset
The dataset contains information about student performance such as study time, attendance, and previous grades.
## Data Preprocessing
- Removed missing values
- Removed duplicate records
- Applied label encoding for categorical variables
- Performed feature scaling
- Split data into training and testing sets
## Models Used
1. Linear Regression
2. Decision Tree Regressor
3. Random Forest Regressor
## Evaluation Metrics
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Error (MAE)
- R² Score
## Result
Among the three models, Random Forest performed the best with the highest R² score.
## Visualization
A bar chart is used to compare the performance of the models.
